WinTclTk TkWrap

Tcl/Tk wrapper for Microsoft Windows
Version 0.2

Tcl/Tk scripts can be wrapped into single-file executables that do not require any Tcl/Tk distributions installed.

There are 3 wrapper files with different package sets:
  • tkwrap.exe - Tcl/Tk and some small packages only
  • xowrap.exe - XOTcl and tDOM
  • fullwrap.exe - all available packages

For in-file help and command line arguments please execute one of the wrapper files.
You can rename the exe files to any other filename (e.g. "wish.exe") and use it as a standard wish shell.
For more information on packages please visit the WinTclTk homepage.
