WinTclTk

Tcl/Tk Distribution for Microsoft Windows
Version 0.3.2

This is a Tcl/Tk distribution for 32-bit Microsoft Windows operating systems.
It includes the following packages:

Tcl/Tk8.4.14
XOTcl1.5.3
tcllib1.9
tklib0.4.1
BWidget1.8.0
tls1.5.0
Tgdbm0.5
Metakit2.4.9.6
mysqltcl3.02
pgtcl1.5.3
mkZiplib1.0
TWAPI1.0.2
ASED3.0.b16
XOTclIDE0.81

The binaries and libraries are compiled using the MinGW distribution with GCC 3.4.5 and stripped.
GDBM 1.8.3 and OpenSSL 0.9.8d are bundled.
The binaries are less efficient than binaries compiled with Microsoft Visual C++. The main idea of this package is to provide a (fully) free enhanced Tcl/Tk package for Microsoft Windows (C).
