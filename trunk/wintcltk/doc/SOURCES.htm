Downloading WinTclTk Sources

Downloading sources automatically:

To download sources automatically you require wget.exe and unzip.exe
Both wget.exe and unzip.exe need to be placed in executable paths (e.g. C:\MinGW\bin or C:\MSYS\1.0\bin)

WGET
http://prdownloads.sourceforge.net/mingw/wget-1.9.1-mingwPORT.tar.bz2
http://xoomer.alice.it/hherold/

UNZIP
http://prdownloads.sourceforge.net/gnuwin32/unzip-5.51-1-bin.zip
http://prdownloads.sourceforge.net/gnuwin32/unzip-5.51-1.exe
