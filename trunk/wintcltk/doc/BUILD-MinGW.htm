Building WinTclTk

Compilation requirements

MinGW MSYS Package, recommended version 1.0.10
http://prdownloads.sf.net/mingw/MSYS-1.0.10.exe

MinGW GCC, recommended from package version 5.1.3
http://prdownloads.sourceforge.net/mingw/MinGW-5.1.3.exe

Perl (required for OpenSSL, all Win32 distributions should work, perl.exe has to be in PATH)
ActivePerl: http://www.activestate.com/Products/ActivePerl
VanillaPerl: http://vanillaperl.com/files/vanilla-perl-5.8.8-build-7.exe

Installer requirements

NSIS Installer, required for creating the exe installer
http://nsis.sourceforge.net/Download

Compilation Instructions

  1. Please read SOURCES.htm (in this directory)
  2. Start the MSYS shell
  3. Change to the distribution directory
  4. To compile all sources, type "make install"
  5. To compile and create the executable installer, type "make"
